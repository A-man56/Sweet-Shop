## âš™ï¸ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/A-man56/Sweet-Shop
cd SweetShop
```

### 2. Backend Setup

```bash
cd backend
npm install
```

Create `.env`:

```
MONGO_URI=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
PORT=5000
```

Run backend:

```bash
npm run dev
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
Create `.env`:

```
VITE_CLOUDINARY_CLOUD_NAME=...
VITE_CLOUDINARY_UPLOAD_PRESET=...

Frontend runs at:

```
http://localhost:5173
